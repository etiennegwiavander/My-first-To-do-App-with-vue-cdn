# My-first-To-do-App-with-vue-cdn
A simple to-do list to write your daily activities. The UI of this application is very easy to understand with the help of tooltips and hard coded and dynamic tips. this application is divided into two sections: 
-Section #1; the To-Do Task section, containing the an array behind the scence that stores the activities to be carried out by the user. Once a user is done with an activity, clicking ones will transfer the activity from this section to the next section.
-section #2; the Completed task section. Containing the activities from section #1. User can interact with information from this section by:
a) clicking ones to restore the task, if an error was made.
b) double clicking to completely remove the task from the DOM.
